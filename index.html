flect — quick start

Flect is a hand-built Astro documentation template with sidebar navigation, instant search, and dark mode, plus an optional terminal-cast player for showing real command output inside your guides. It has no dependency on Starlight or any other docs framework — every component here is plain Astro, and it's yours to read, edit, and own outright.

No framework, no lock-in.

Flect is a set of plain Astro components and pages you can read end-to-end — not a plugin system to configure around.

No framework lock-in

Every page and component is plain Astro — no Starlight, no hidden abstraction to work within.

Instant search, built in

Pagefind indexes your docs at build time, with an auto-generated table of contents and sidebar that stay in sync with your content.

Built for CLI and dev-tool docs

An optional terminal-cast player embeds real asciinema recordings, so guides can show actual command output instead of static code blocks.

Deploy in one push

A GitHub Pages workflow and pull request previews are already wired up — push to main and the docs are live.

Start in one command.

Clone the template, install dependencies, and the dev server is running.

Clone and runNo CLI to install first, no scaffolding step — just a repository you clone and run.
git clone https://github.com/shravanngoswamii/flect.git
cd flect
npm install
npm run dev

FAQ

Quick answers for search engines and humans.

What is Flect?

A hand-built Astro documentation template with search, theming, and sidebar navigation ready to use out of the box.

Do I need Starlight or another docs framework?

No. Flect has no dependency on Starlight or any other docs framework — it's plain Astro components and pages.

Can I remove the terminal-cast player?

Yes. It's entirely optional — delete the component and its recording if you don't need it.

How do I deploy it?

A GitHub Pages workflow is included, so pushing to main builds and deploys the site automatically.
